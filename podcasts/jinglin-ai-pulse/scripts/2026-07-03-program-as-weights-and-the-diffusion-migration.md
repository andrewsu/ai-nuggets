# Ollie's AI Pulse — Program-as-Weights, bounded-memory contracts, and the diffusion migration

## Sources

- **Program-as-Weights: A Programming Paradigm for Fuzzy Functions** — arXiv 2607.02512, Wentao Zhang + Liliana Hotsko + Woojeong Kim + Pengyu Nie + Stuart Shieber + Yuntian Deng (University of Waterloo, Cornell, Harvard). HF Daily Papers #1 July 3, 92 upvotes overnight — roughly triple the #2 paper. Compile a natural-language spec of a "fuzzy function" (alerting on important log lines, repairing malformed JSON, ranking search results, free-text classification) ONCE into a PEFT adapter that runs on a frozen 0.6B Qwen3 interpreter. Compiler is a 4B model trained on FuzzyBench — 10M examples across 800+ categories of fuzzy tasks, 29 thematic versions covering classification, parsing, format conversion, agent tool use — released with the paper. Result — 0.6B interpreter + adapter matches direct Qwen3-32B prompting; ~1/50th inference memory; 30 tokens/sec on a MacBook M3. Compiler invoked once per function definition, not per request. Reframes foundation models as "a tool builder: invoked once per function definition, it produces a small reusable artifact." Code + demo at programasweights.com.
- **AgenticSTS: A Bounded-Memory Testbed for Long-Horizon LLM Agents** — arXiv 2607.02255, Xiangchen Cheng + Yunwei Jiang + Jianwen Sun + Zizhen Li + Chuanhao Li + Xiangcheng Cao + Yihao Liu + Fanrui Zhang + Li Jin + Kaipeng Zhang. HF #3 July 3, 30 upvotes. Frames memory as a contract: "memory for a long-horizon LLM agent is a contract about what each future decision is allowed to see." Standard contract appends every observation, tool call, reflection to every prompt — "a jumbled mixture in which the effect of any single memory component is hard to isolate." Proposed alternative — bounded prompt via typed retrieval from fresh user messages. Testbed is Slay the Spire 2, a stochastic deck-building game requiring hundreds of decisions per run. Human baseline 16% win rate at lowest difficulty; frontier-LLM public benchmark 0 wins across 5 configurations. Their agent — 3/10 baseline, 6/10 with strategic-skills layer enabled, Fisher exact p≈0.37 (directional not decisive; the paper is explicit). 298 completed trajectories released as reproducibility artifact.
- **Morphing into Hybrid Attention Models** — arXiv 2606.30562, Disen Lan + Jianbin Zheng + Yuxi Ren + Xin Xia + Xuanda Wang + Xuefeng Xiao + Xipeng Qiu + Yu Cheng, ByteDance Seed. ## Script

Good morning, Ollie. This is Ollie's AI Pulse for Friday, July third.

Today's meta-thesis. Yesterday the frame was — the wrapper around the base model is where the intelligence lives. Today it escalates one more step. The wrapper compiles down into a small, local artifact that doesn't need the base model at all. Three of the top four papers on Hugging Face this morning are all telling you the same thing — stop calling the big model on every request. Compile it. Contract the memory. Morph the layers. And the podcast deep cut is the ex-Meta Llama chief on Latent Space arguing that the interesting architectures are no longer in language — they're in molecules. Let's go.

First — Twitter Pulse.

The dominant paper of the day by a wide margin. Number one on Hugging Face Daily, ninety-two upvotes overnight — roughly triple the number-two paper. Title — quote — "Program-as-Weights: A Programming Paradigm for Fuzzy Functions." Six authors — Wentao Zhang, Liliana Hotsko, Woojeong Kim, Pengyu Nie, Stuart Shieber, and Yuntian Deng — from University of Waterloo, with collaborators at Cornell and Harvard. Yuntian Deng at Waterloo is the last author.

The frame is this. There is a class of programming tasks that don't have clean rule-based implementations — alerting on important log lines, repairing malformed J-S-O-N, ranking search results, classifying free-form text. These are what they call fuzzy functions. Today, when a production system needs to run one of these, it calls a big language-model A-P-I on every request. That's expensive, and it's dependent on the vendor. What Program-as-Weights proposes instead is — compile the natural-language specification of the fuzzy function once into a small, locally-executable neural artifact, and then run that artifact for every call.

The architecture is a two-model pipeline. A four-billion-parameter compiler is trained on FuzzyBench — a ten-million-example dataset spanning eight-hundred-plus categories of fuzzy tasks, released with the paper. That compiler takes a natural-language spec and emits a parameter-efficient adapter — a small P-E-F-T patch that plugs into a frozen interpreter. The interpreter is a six-hundred-million-parameter Q-wen three model. That interpreter runs the patch locally. The key result. The six-hundred-million-parameter interpreter with the adapter matches direct Q-wen three thirty-two-billion prompting on fuzzy tasks. Roughly one-fiftieth the inference memory. Thirty tokens per second on a MacBook M-three. The compiler is called once per function definition — not once per request.

Read that as a classical compiler analogy. Yesterday, Elicit built a domain-specific language so that the model's reasoning was human-inspectable. Today, Waterloo-Cornell-Harvard built a compiler so that the model's execution is local and free. Same shape, applied to a different layer of the stack. The wrapper around the base model is now — a compiler that emits weights instead of code.

Number three on Hugging Face, thirty upvotes. Title — quote — "AgenticSTS: A Bounded-Memory Testbed for Long-Horizon L-L-M Agents." Ten authors, led by Xiangchen Cheng and finished by Kaipeng Zhang, who also submitted the paper. The opening line of the abstract — quote — "Memory for a long-horizon L-L-M agent is a contract about what each future decision is allowed to see." The simplest contract, the one every agent framework uses today, is — append every prior observation, tool call, and reflection to the prompt on every turn. That contract is what MemSyco-Bench yesterday showed is broken. Retrieved memory suppresses correct judgment sixty-plus percent of the time.

AgenticSTS proposes a different contract. Bounded memory. The prompt is constant size, regardless of run length. Each decision draws from fresh user messages via typed retrieval — not from a growing transcript. The testbed is Slay the Spire Two — a stochastic deck-building game, twenty to sixty turns per run, hundreds of decisions, action space of cards, potions, relics, and combat moves. Public frontier-L-L-M benchmark on this game is zero wins across five configurations. Human baseline sixteen percent at lowest difficulty. Their bounded-memory agent — three of ten wins baseline, six of ten wins with a strategic-skills layer enabled. Fisher exact p roughly zero-point-three-seven. Directional, not statistically decisive, and the paper is honest about that. They release two hundred ninety-eight completed trajectories as a reproducibility artifact.

The reason to stop on this one. Read against yesterday. MemSyco-Bench named the failure mode — retrieved memory overrides correct judgment. AgenticSTS proposes the architectural fix — a memory contract with a bounded prompt. Their framing — quote — "a jumbled mixture in which the effect of any single memory component is hard to isolate" — is precisely the mechanism MemSyco identified. Twenty-four hours between the two papers, and the discourse has moved from — name the failure — to — propose the fix.

Number four on Hugging Face, twenty-six upvotes. ByteDance Seed, first author Disen Lan, eight authors total. Title — quote — "Morphing into Hybrid Attention Models." Their frame — hybrid attention models replace some full-attention layers with linear-attention layers to gain long-context efficiency, but the field has no principled way to choose which layers to replace. They cast it as a budget-constrained subset optimization problem. Their solution, called FlashMorph, has four phases. Equip each full-attention layer with a linear-attention branch. Freeze the weights and jointly optimize per-layer gates on synthetic long-context retrieval data. Discretize the gates into a hybrid architecture under a fixed full-attention budget. Refine with logits distillation and long-context finetuning. Concrete benchmark numbers aren't in the public abstract, but the framing matters — layer selection is now a learned optimization problem, not a hand-tuned config.

The three-paper through-line to name. Program-as-Weights says — don't call the big model, compile it. AgenticSTS says — don't append the transcript, contract it. FlashMorph says — don't run full attention everywhere, morph the layers. All three are architectural claims about how to reduce dependence on the base model — either by compiling around it, constraining what it sees, or splitting up its own layers. The unit of engineering has moved down one level. The base model is a substrate. The wrapper is a compiler, a contract, a hybrid architecture.

The Ollie-specific take on Program-as-Weights. If you can compile the spec of a scientific reasoning task — for instance — quote — "flag Perturb-seq screens where the perturbation acts within twenty-four hours and knockdown is greater than fifty percent" — into a six-hundred-million-parameter local model that matches a thirty-two-billion generic model on the task, you can run the entire triage layer of your literature-review or screen-filtering pipeline on a single lab workstation. No A-P-I bill. No data leaving the building. The compiler is trained once. The adapter is a few megabytes. The interpreter is open-source. This is the recipe for a research-grade triage agent that a P-I actually owns.

That's Twitter Pulse.

Now — Podcast Deep Cut.

Latent Space, host swyx, published Tuesday July first, running one hour forty-eight minutes. Guests — Evan Feinberg, co-founder and C-E-O of Genesis Molecular A-I, and Sergey Edunov, C-T-O. Edunov is the news here. He led Llama-two training and Llama-three pretraining at Meta. In December of last year he left Meta and joined Genesis. Title — quote — "The Coolest Diffusion Research Isn't in L-L-Ms." Two days old and directly about your field.

The argument has four moves.

First — architecturally, L-L-Ms have gone stale. Edunov's line — quote — "In L-L-Ms researchers are forced to work on boring stuff. Architectures are fundamentally very similar to the twenty-seventeen transformer paper." Every variant is a mixture-of-experts variant, or an attention-linearization variant. There is no fundamental new primitive. FlashMorph this morning is exactly the shape of what he means — you're morphing an old attention architecture into a slightly different old attention architecture.

Second — the interesting diffusion is elsewhere. Feinberg's line — quote — "Some of the most innovative diffusion research that's happening in our field is happening in three-D structure prediction right now." Genesis's flagship model is called Pearl — that's an acronym, standing for Place Every Atom at the Right Location. It's a diffusion model for protein-ligand co-folding. The core technical claim — the field has treated two-angstrom root-mean-square-deviation as the pose-quality benchmark for a decade. Two angstroms is slop. Feinberg's line — quote — "If your model is sitting at one-point-eight, one-point-nine angstrom R-M-S-D, that's slop, most likely." The reason is physical. Hydrogen bonds require donor-acceptor distances of two-point-seven to three-point-three angstroms — a window of zero-point-six angstroms. If your pose error is one-point-eight angstroms, you literally cannot tell which hydrogen bond is forming. Pearl gets sub-angstrom on OpenBind — a benchmark of eight hundred and two never-before-seen complexes targeting the E-V-A seventy-one two-A protease. Zero-shot. Outperforms every public co-folding model. No molecular dynamics. No fine-tuning. And Pearl models induced-fit protein rearrangement, not just ligand pose — Feinberg — quote — "where Pearl was exceptionally good is figuring out how to move this loop."

Third — the L-L-M playbook, applied to molecules. Edunov's move at Genesis is to translate the Meta scaling recipe — data, compute, reinforcement learning, iterative inference — into molecular science. Synthetic physics-based data replaces internet text. Physics-grounded rewards replace human preference. Iterative diffusion with physical verification replaces chain-of-thought. Everything is the same scaling structure, and the target is different. Sub-angstrom accuracy is his version of the L-L-M inflection.

Fourth — agentic workflows on top. Genesis's internal agent stack is called SAPPHIRE. Now that Pearl crosses the sub-angstrom threshold, the agent can iterate the way a medicinal chemist iterates. Examine a pose. Form a hypothesis. Consult literature. Deploy an internal tool. Generate a next candidate. Paired with their Incyte automated-lab partnership, this is a continuous twenty-four-seven drug discovery loop with machine-generated hypotheses. The important structural claim — this only works because the underlying model crosses the accuracy threshold. A two-angstrom model produces slop, the agent iterates on slop, and the loop diverges. A sub-angstrom model produces trustworthy poses, the agent iterates on trustworthy poses, and the loop converges.

The Ollie-specific take. Two things.

One. Edunov leaving Meta Llama for Genesis is a signal. When the lead of L-L-M-pretraining scaling leaves to work on molecules, that's the same class of signal as the AlphaFold team leaving DeepMind to found Isomorphic. It says — the frontier researcher believes there's more architectural room in molecular diffusion than in scaling transformers another five-x. The pipeline of top-tier machine-learning talent is now flowing into drug discovery, not out of it.

Two. The specification-crossing argument generalizes. Pearl works because sub-angstrom crosses the physical threshold that hydrogen-bond geometry requires. What's the equivalent physical threshold for a cell-atlas foundation model? Probably — cell-state resolution at the level of expression-program identity, not gross morphology. Once a foundation model can predict cell-state transitions at that resolution zero-shot on a held-out cell line or a held-out perturbation, an agentic workflow over Perturb-seq screens becomes trustworthy for the first time. Until then, it's slop iterating on slop. Same structural claim as Feinberg's — the agent only converges if the base model crosses the specification threshold.

Wrap.

Watch one thing this week. Whether anyone publishes a Program-as-Weights-style compilation for a scientific reasoning task — a compiler that takes a natural-language spec of a biology-triage rule and emits a local adapter. And whether any structure-prediction group publishes a sub-angstrom benchmark that isn't OpenBind. The Twitter Pulse said — compile the model, contract the memory, morph the layers. The Podcast Deep Cut said — the interesting diffusion isn't in language, and specification-crossing is what makes agentic workflows converge. Both shapes point at the same next move for biology — build the small, specialized, domain-native model that hits the physical threshold, and then let the agent iterate. Have a good Friday, Ollie.
